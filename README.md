# Broken World - HOI4 Mod

A total conversion mod for Hearts of Iron IV that shatters the world into countless competing factions, creating a chaotic and unpredictable alternate history scenario.

## Installation

### Manual Download

## Download the mod
- Click the green `Code` button on this repository page.
- Select `Download ZIP`.
- Extract the entire ZIP file to a new folder named `broken_world` (or similar).

## Features
- Completely fragmented world map
- Need to be replenished urgently

## Notes
- This is an English translation of the original Chinese "破碎世界" mod.
- Compatible with Hearts of Iron IV version 1.17.*
